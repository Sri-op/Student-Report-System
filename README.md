# Student Report System

## Project Description
The Student Report System is a Python program that allows users to enter the marks of multiple students and generates a report. It calculates the average marks, highest marks, lowest marks, and the number of students who passed.

## Features
- Accepts marks for N students
- Stores marks in a list
- Calculates average marks
- Finds highest and lowest marks
- Counts students who passed (marks >= 40)
- Displays a complete student report

## Technologies Used
- Python 3
- Lists
- Functions
- Loops
- Conditional Statements

## Concepts Covered
- User Input
- Lists
- Functions
- Loops
- Conditions
- Built-in Functions (sum(), max(), min())

## How to Run
1. Install Python 3.
2. Save the program as `student_report.py`.
3. Open Command Prompt or Terminal.
4. Run the program using:
   ```
   python student_report.py
   ```
5. Enter the number of students and their marks.

## Sample Input
```
Enter number of students: 5
Enter marks of student 1: 80
Enter marks of student 2: 65
Enter marks of student 3: 45
Enter marks of student 4: 30
Enter marks of student 5: 90
```

## Sample Output
```
------ Student Report ------
Marks: [80, 65, 45, 30, 90]
Average: 62.0
Highest: 90
Lowest: 30
Passed Students: 4
```
